# CODSOFT Python Programming Internship

## Overview
This repository contains the Python programming tasks completed as part of the **CodSoft Python Programming Internship**. Each project demonstrates fundamental Python concepts such as loops, functions, conditional statements, user input, and modules.

---

## Task 1: To-Do List Application

### Description
A command-line application that helps users manage their daily tasks efficiently.

### Features
- Add new tasks
- View all tasks
- Update existing tasks
- Mark tasks as completed
- Delete tasks
- Simple menu-driven interface

**Technology Used:** Python

---

## Task 2: Simple Calculator

### Description
A basic calculator that performs arithmetic operations based on the user's choice.

### Features
- Addition
- Subtraction
- Multiplication
- Division
- Division-by-zero handling
- Easy-to-use interface

**Technology Used:** Python

---

## Task 3: Rock Paper Scissors Game

### Description
A command-line implementation of the classic Rock, Paper, Scissors game where the user plays against the computer.

### Features
- Random computer selection
- Winner determination
- Score tracking
- Multiple rounds
- Play again option
- User-friendly interface

**Technology Used:** Python
